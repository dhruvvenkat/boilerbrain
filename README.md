# BoilerBrain

Writing boilerplate and setting up scaffolding for your backend projects is inconvenient, BoilerBrain simplifies the process

Input project descriptions into the CLI and watch an AI agent generate a basic project spec, make architectrual decisions, recommend a tech stack, and generate all the boilerplate code + endpoints for you!!
